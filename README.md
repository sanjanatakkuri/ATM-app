 ATM Web Application 

A simple ATM Web Application built using Python, Flask, HTML and CSS.  
This project simulates basic ATM operations such as creating an account, checking balance, depositing, withdrawing, updating details, and deleting an account.

 Features

- Create a new bank account  
- Check account balance  
- Deposit money  
- Withdraw money  
- Update account details  
- Delete account  
- User-friendly web interface  


Technologies Used

- Python
- Flask
- HTML
- CSS
- MySQL (or database used in your project)
